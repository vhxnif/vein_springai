# vein_springai

Spring AI 知识库，供 [vein](https://github.com/vhxnif/vein) 使用。

## 连接

```bash
vein project --connect /path/to/vein_springai
```

## 内容

| 目录 | 说明 | 数量 |
|------|------|------|
| `springai2.0/` | Spring AI 2.0 参考文档 | 106 篇 |
| `posts/` | Spring Blog 文章（Spring AI 主题） | 50 篇 |

## 更新

```bash
# 更新 Spring AI 参考文档
./download_springai_docs.sh

# 更新 Spring Blog 文章
# 使用 springblog skill
```
