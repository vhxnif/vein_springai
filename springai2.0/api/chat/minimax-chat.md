# MiniMax Chat

As of Spring AI 2.0 and as recommended in the MiniMax documentation, MiniMax models should be accessed via the Anthropic support with `spring.ai.anthropic.base-url=https://api.minimax.io/anthropic` and the model customized to use one the MiniMax models, for example `spring.ai.anthropic.chat.model=MiniMax-M3`.
